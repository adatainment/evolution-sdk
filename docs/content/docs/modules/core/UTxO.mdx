# models

## UTxOSet (type alias)

A set of UTxOs with efficient lookups and set operations.
Uses Effect's HashSet for automatic deduplication via Hash protocol.

**Signature**

```ts
export type UTxOSet = HashSet.HashSet<UTxO>
```

Added in v2.0.0
